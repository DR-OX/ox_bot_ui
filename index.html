Blender Add-on Components - ox_bot

This page displays a list of components used in Blender add-on development.

# Component Description Blender API Equivalent
1 Panel Blender's UI section for tools and settings, where add-ons add functionalities. bpy.types.Panel
2 Column Arranges elements vertically within panels. layout.column()
3 Row Arranges elements horizontally within panels. layout.row()
4 Dropdown Provides a list of options for users to select. layout.prop() or layout.menu()
5 Button A clickable element that triggers a command. layout.operator()
6 Label Text element used to provide information within panels. layout.label()
